Wireless Gesture Recognition System

2026
  • Built a wireless gesture control system using an MPU6050 accelerometer/gyroscope interfaced over I2C with an ESP8266 microcontroller for real-time motion capture.
  • Engineered wireless data transmission from the ESP8266 to a Raspberry Pi 4 to process real-time motion data.
  • Designed system block diagrams and control logic to map hand gestures to LED state changes.

IoT-Based Weather Monitoring System

2025
  • Integrated Arduino Uno with DHT11 and BMP180 sensors over I2C/digital protocols, with live readings on a JHD162A 16×2 LCD.
  • Engineered UART-based serial communication between Arduino and an ESP8266 Wi-Fi module to transmit readings to a remote server.
  • Implemented temperature, humidity, and pressure sensing with robust Embedded C logging routines.
Arduino UnoDHT11BMP180UART

Maze-Solving Robot

2024
  • Designed an autonomous maze-solving robot using Arduino Uno, three HC-SR04 ultrasonic sensors, and an L293D dual H-bridge motor driver.
  • Implemented real-time obstacle avoidance and directional logic from ultrasonic feedback.
  • Programmed sensor polling and motor control in Embedded C/C++, iterating on navigation accuracy.
